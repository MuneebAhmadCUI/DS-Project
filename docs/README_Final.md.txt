## Project Summary
RailVerse is a console-based booking system designed to manage train reservations efficiently. It uses a **Singly Linked List** to store passenger records dynamically, allowing for real-time additions and cancellations without fixed memory limits.

## Key Features
1.  **Dynamic Booking:** Add unlimited passengers (Add Node).
2.  **Cancellation System:** Remove bookings by ID (Delete Node).
3.  **Search Engine:** Find passenger details instantly (Linear Search).
4.  **Live Manifest:** View all active bookings (Traversal).

## How to Run
1.  Open `src/main.cpp` in Dev-C++ or VS Code.
2.  Compile and Run.
3.  Use the numeric menu to navigate.

## Data Structure Used
* **Struct:** For Passenger details (ID, Name, Train).
* **Singly Linked List:** For managing the database in RAM.
